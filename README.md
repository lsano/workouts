# Workouts

A Next.js workout tracking application.

## Getting Started

First, install dependencies:

```bash
npm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

```bash
npm run build
npm run start
# or
bun run build
bun run start
```

## Linting

```bash
npm run lint
# or
bun run lint
```
