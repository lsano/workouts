# Workouts — Watch Plan Editor

A Next.js workout tracking app with a plan editor and Apple Watch data entry.

## Getting Started

First, install dependencies:

```bash
npm install
# or
bun install
```

Then, run the development server:

```bash
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

```bash
npm run build
npm run start
# or
bun run build
bun run start
```

## iOS / Capacitor

Build and open in Xcode:

```bash
npm run cap:build-ios
# or
bun run cap:build-ios
```

Other Capacitor commands:

```bash
# Add iOS platform (first-time setup)
npm run cap:init-ios

# Sync web build to native project
npm run cap:sync

# Open existing Xcode project
npm run cap:open-ios
```

## Linting

```bash
npm run lint
# or
bun run lint
```
